On Windows:
Copy .exes of firefox and your version of chrome to System32 or add them to PATH variable

On Linux:
Copy files of firefox and your version of chrome to /usr/bin or /usr/local/bin or add them to PATH variable

The number in the directory name is the version of chrome for which they are made

Driver source:
Chrome: https://chromedriver.chromium.org/downloads
https://github.com/mozilla/geckodriver/releases
